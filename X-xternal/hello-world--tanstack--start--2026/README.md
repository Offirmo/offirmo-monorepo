Install @tanstack/react-start and set up its Agent Skills:

1. Install the package using the project's package manager:
   @tanstack/react-start

2. Ensure @tanstack/intent is installed:
   @tanstack/intent

3. Discover and install the skills shipped with this package:
   npx @tanstack/intent install

This will detect all Agent Skills in @tanstack/react-start and configure them for your coding agent.
